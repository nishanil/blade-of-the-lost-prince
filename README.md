# Blade of the Lost Prince

A browser-based side-scrolling action game inspired by the original Prince of Persia. Fight guards, avoid traps, and escape the dungeon.

**Play now: https://nishanil.github.io/blade-of-the-lost-prince/**

## Controls

### Desktop
| Key | Action |
|-----|--------|
| Arrow Keys | Move |
| Shift + Arrow | Walk (slow) |
| Space | Jump |
| Enter | Attack |
| S | Block |

### Mobile
On-screen touch controls appear automatically on touch devices:
- **D-Pad** — Move left/right
- **JUMP** — Jump
- **ATK** — Attack
- **BLK** — Block

## Features

- Authentic dungeon aesthetic with stone walls, torches, and pillars
- Detailed character sprites with animation (run, walk, jump, attack, block)
- Guard AI with patrol, chase, attack, and block behaviors
- Pits with spikes, elevated platforms, and an exit to escape
- Responsive layout with mobile touch support (portrait 9:16)
- Single HTML file, no dependencies
